2. Description du jeu

Guess Number est une application mobile de jeu de prédiction de numéros, conçue pour le divertissement entre amis. Le principe est simple :

  • Un numéro gagnant est tiré à chaque round
  • Les joueurs qui ont parié sur ce numéro remportent des gains
  • Les gains sont crédités automatiquement après chaque round
  • Les joueurs peuvent retirer leurs gains sur leur Mobile Money

Guess Number est un jeu de divertissement. Les résultats sont aléatoires. Il n'existe aucune stratégie garantissant des gains.
